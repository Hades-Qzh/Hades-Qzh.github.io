Jupyter Notebook与Markdown知识点汇总

这篇博客是有关Jupyter Notebook与Markdown的
1>Markdown：
（1）换行与段落：
写完第一行的内容后，与在word上打字相比，不是简简单单的Enter就能换行，而要连续空两格。
（2）标题：
在Markdown中，标题总共有六级。每增加一个”#”，字体大小就会降一级。敲重点：“#”后一定要加空格
